# IPL Real-Time Analytics Platform

An end-to-end IPL real-time analytics platform built using Python, Snowflake, DBT, and Snowpark. The project processes and analyzes IPL data, performs data transformations, and provides interactive dashboards to generate insights into team performance, player statistics, and match trends.

## Technologies Used
- Python
- Snowflake
- DBT
- Snowpark
- SQL
- Jupyter Notebook

## Features
- Real-time IPL analytics
- Data transformation pipelines
- Interactive dashboards
- Player and team performance analysis
- Cloud-based data warehousing
